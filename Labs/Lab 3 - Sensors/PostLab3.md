# Lab 3: Sensors - Individual
## Description

Build a system that measures something interesting and reports the data somehow.

## ESP32 - Camera Module
- For my Final Project, I was able to make an ESP32 camera module system that detects light/color intensity and translates it to positional data, tracking the position of a brightly coloured object (pink highlighter).
- It measures the location of the object within the camera frame in pixel x,y coordinates.
- The physical quantities measured are the object position in x,y coordinates and the motion over time or the change in position across frames.
