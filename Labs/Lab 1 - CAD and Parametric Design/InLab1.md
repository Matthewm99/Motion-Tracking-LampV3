# Lab 1: Parametric CAD and Laser Cutting
## Description

For the first in-lab assignment, our group designed, laser cut and assembled an interlocking box using parametric CAD in SolidWorks. The box is composed of two unique desings, with one design being used for the top and bottom sides, and the other design being used for the four side pieces.

## What We Learned
- Parametric Drawing - Parametric drawing was the main takeway from this lab. We learned how to parameterize, defining our dimensions relative to one driving dimension rather than fixed values.
- Inkscape and SolidWorks Best Practices - We learned some valuable practices for using these applications such as stroke colour conventions (red for laser cutting, blue for engraving), removing watermarks and exporting and importing between applications.
- Tolerances - One mistep that we took was defining our tolerances as percentages. This is flawed as the percentage is dependent on the size and should rather be an set value.
